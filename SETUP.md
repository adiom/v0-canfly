# Canfly - Артхаусное издательство комиксов и книг

Полнофункциональный сайт вселенной комиксов Canfly с книгами, персонажами, онлайн-чтением и магазином.

## Функциональность

✅ **Главная страница** - Слайдер с избранными книгами/комиксами
✅ **Каталог персонажей** - Профили героев с описаниями и граф взаимосвязей
✅ **Онлайн-читалка** - Просмотр страниц книг прямо на сайте
✅ **Магазин** - Каталог всех изданий с ценами и описаниями
✅ **Корзина** - Добавление товаров в корзину (localStorage)
✅ **Оформление заказов** - Форма заявки с сохранением в БД
✅ **Админ-панель** - Управление книгами, персонажами и просмотр заказов

## Технический стек

- **Frontend**: Next.js 16, React 19, TypeScript, Tailwind CSS v4
- **Backend**: Next.js API Routes
- **База данных**: Neon/Vercel Postgres
- **UI**: shadcn/ui компоненты
- **Аутентификация**: Простая система для администраторов

## Установка и запуск

### 1. Клонирование проекта

```bash
git clone <repository-url>
cd canfly
```

### 2. Установка зависимостей

```bash
pnpm install
```

### 3. Переменные окружения

Создайте файл `.env.local`:

```env
DATABASE_URL=postgres://user:password@host/db?sslmode=require
NEXT_PUBLIC_BASE_URL=http://localhost:3000
ADMIN_PASSWORD=admin123
```

### 4. Запуск dev-сервера

```bash
pnpm dev
```

Откройте [http://localhost:3000](http://localhost:3000)

## Использование админ-панели

### Вход в админку

1. Перейдите на страницу `/admin/login`
2. Используйте учетные данные:
   - **Email**: `admin@canfly.local`
   - **Пароль**: `admin123` (или значение `ADMIN_PASSWORD` из env)

### Управление содержимым

**Книги/Комиксы:**
- Просмотр всех изданий
- Добавление новых книг (с обложкой, описанием, ценой)
- Редактирование существующих
- Удаление

**Персонажи:**
- Просмотр всех персонажей
- Добавление новых героев (с аватаром, био, описанием)
- Управление взаимосвязями между персонажами
- Удаление

**Заказы:**
- Просмотр всех полученных заказов
- Отслеживание статуса (pending, confirmed, shipped, completed)
- Просмотр деталей заказа (товары, контакты, адрес)

## Структура БД

### Таблицы

**books** - Книги/комиксы
- id, title, slug, type (comic/book/audiobook)
- description, cover_image, preview_pages (JSONB)
- price, is_featured, display_order
- external_links (ссылки на внешние магазины)

**characters** - Персонажи
- id, name, slug, avatar, bio, full_description
- abilities (JSONB массив способностей)

**character_relationships** - Связи между персонажами
- character_id, related_character_id, relationship_type, description

**book_characters** - Many-to-many связь книг и персонажей

**orders** - Заказы
- customer_name, customer_email, customer_phone, customer_address
- items (JSONB с деталями товаров)
- total, status, notes

**admins** - Администраторы (email whitelist)

## API Endpoints

### Публичные

- `GET /api/books` - Получить все книги
- `GET /api/books?featured=true` - Получить избранные книги
- `GET /api/characters` - Получить всех персонажей
- `GET /api/characters/[slug]` - Получить персонажа и его связи
- `POST /api/orders` - Создать новый заказ

### Администраторские

- `POST /api/admin/login` - Вход администратора
- `GET /api/admin/orders` - Получить все заказы

## Кастомизация

### Обновление образа наружности сайта

Все цвета и стили в `app/globals.css` и компонентах используют Tailwind CSS v4.
Основная палитра: Фиолетовый (#a855f7) и розовый (#ec4899) как акценты на темном фоне.

### Добавление новых книг

1. Войдите в админ-панель
2. Нажмите "Добавить книгу"
3. Заполните форму:
   - Название, slug (для URL)
   - Тип (комикс/книга/аудиокнига)
   - Описание
   - Обложка (URL или загрузка)
   - Цена (в копейках: 59999 = 599.99 ₽)
   - Ссылки на внешние магазины (Ozon, Litres и т.д.)
   - Страницы для чтения (URLs изображений)

### Добавление персонажей

1. В админ-панели выберите "Персонажи"
2. Нажмите "Добавить персонажа"
3. Заполните данные:
   - Имя и slug
   - Аватар (изображение)
   - Краткое описание (bio)
   - Полное описание
   - Способности (массив)
4. После создания можете добавлять связи с другими персонажами
