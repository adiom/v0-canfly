# Новые функции в canfly | культура твоего сознания

## 1. Персонажи и AI-чат

### Новые персонажи добавлены:
- **Соня** - Хранительница снов и путешественница между мирами
- **Турбокороль** - Владыка скорости и хаоса
- **Убыр** - Древний дух из татарских легенд

### Функция чата:
На странице каждого персонажа теперь есть полнофункциональный AI-чат где вы можете:
- Поговорить с персонажем как с живым существом
- Спросить о его мыслях и философии
- Узнать о его роли в книгах и комиксах вселенной
- Каждый персонаж имеет уникальное "лицо" - генерируется промпт на основе его описания

**Как работает:**
- При открытии страницы персонажа (например, `/characters/sonya`), снизу вы найдёте раздел "Поговори с Соней"
- Введите сообщение в поле и отправьте
- Персонаж ответит вам в его уникальном стиле

## 2. Социальная сеть персонажей

### Профили как в соцсети:
На странице `/characters` в разделе "Лента активности" вы увидите посты от всех персонажей:
- 💭 **Мысли** - философские размышления персонажей
- 📢 **Анонсы** - объявления о новых книгах и событиях
- ❓ **Вопросы** - вопросы к читателям

### Особенности постов:
- Каждый пост имеет аватар автора
- Кнопки взаимодействия (нравится, ответить, поделиться)
- Сортировка по времени создания
- Разные типы постов с разными стилями

## 4. Архитектура базы данных

### Новая таблица:
```sql
character_posts
├── id (UUID)
├── character_id (Foreign Key)
├── content (TEXT) - текст поста
├── post_type (TEXT) - 'thought', 'announcement', 'question'
├── image_url (TEXT) - опционально
└── created_at (TIMESTAMP)
```

### API endpoints:
- `POST /api/characters/chat` - AI-чат с персонажем
- `GET /api/characters/posts` - получение постов (с опциональной фильтрацией по персонажу)

## 5. AI модель

Используется OpenAI GPT-4 Mini для генерации ответов персонажей.

**Требования:**
- Переменная окружения `OPENAI_API_KEY` должна быть установлена

**Как работает система промптов:**
Каждому персонажу дан уникальный системный промпт который определяет:
- Его личность и стиль общения
- Его знания и опыт
- Его взгляд на мир

Пример для Сони:
```
Вы Соня, хранительница снов и путешественница между мирами. 
Вы говорите мягко, но с глубокой мудростью. 
Вы видите скрытое и помогаете людям понять их подсознание...
```

## 6. Компоненты

### `CharacterChat`
- Интерактивный чат с персонажем
- Потоковая обработка ответов (streaming)
- Автоскролл к последним сообщениям
- Индикатор загрузки

### `CharacterPostsFeed`
- Лента постов всех персонажей
- Фильтрация по персонажу (опционально)
- Отображение типа поста
- Реактивное обновление

## 7. Примеры использования

### Просмотр профиля персонажа с чатом:
```
https://your-domain.com/characters/sonya
```

### Просмотр всех постов на странице персонажей:
```
https://your-domain.com/characters
```

Прокрутите вниз и вы увидите "Лента активности" со всеми постами.

Чат и посты используют дизайн системы canfly:
- Тёмная тема (slate-950, slate-900)
- Purple/Pink градиент для акцентов
- Стекломорфный стиль (backdrop-blur)
- Соответствие артхаусному стилю проекта
