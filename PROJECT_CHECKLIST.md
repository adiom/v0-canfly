## 📝 Примечания

### Тестовые учётные данные
- **Email админа**: `admin@canfly.local`
- **Пароль админа**: `admin123`

### Примечание о коммерции
- Корзина использует **localStorage** для демонстрации
- Заказы сохраняются в Supabase БД с статусом "pending"
- Платежи не интегрированы (можно добавить Stripe позже)

### Переменная окружения ADMIN_PASSWORD
- Используется при входе в админ-панель
- В продакшене используйте Supabase Auth вместо этого
